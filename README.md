# deesl

[![Coverage](https://codecov.io/gh/garvbox/deesl/graph/badge.svg)](https://codecov.io/gh/garvbox/deesl)

A Rust web application for tracking vehicle fuel consumption with a Vue.js frontend.

## Development

```bash
# Run tests
cargo t

# Run tests with output
cargo tc

# Run linter
cargo lint

# Generate coverage report
cargo cov

# Open coverage report
cargo cov-open
```

## Test Coverage

Coverage reports are automatically generated on each pull request and posted as comments.
