# deesl

[![Coverage](https://codecov.io/gh/garvbox/deesl/graph/badge.svg)](https://codecov.io/gh/garvbox/deesl)

A Rust web application for tracking vehicle fuel consumption with a Vue.js frontend.

## Development

```bash
# Run tests
cargo t

# Run tests with output
cargo tc

# Run linter
cargo lint

```
